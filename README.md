# Linear Algebra Library

A reusable linear algebra engine designed to solve mathematical problems with step-by-step explanations.

## Planned Features  

- Vector operations  
- Matrix operations 
- Determinants  
- Matrix inverses  
- Gaussian elimination  
- LU decomposition  
- Eigenvalues and eigenvectors  
- Step-by-step solution
